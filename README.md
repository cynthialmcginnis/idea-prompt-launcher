# IDEA Framework — AI Lesson Plan Builder

A one-click prompt launcher for designing pedagogically sound lessons with AI.

Built on the **IDEA Framework** (Identify · Design · Evaluate · Adapt) — a structured four-step method grounded in seven adult learning theories: Bloom's Taxonomy, Andragogy, Active Learning, Kolb's Experiential Cycle, Constructivism, Fink's Taxonomy, and UDL.

---

## How to Use

1. Open the [Prompt Launcher](https://cynthialmcginnis.github.io/idea-prompt-launcher/) (or your GitHub Pages URL once enabled)
2. Click **Copy Prompt A** and paste into Microsoft Copilot
3. Answer Copilot's 11 questions about your course
4. Click **Copy Prompt B** and paste into the same chat window — Copilot generates your lesson plan
5. Click **Copy Prompt C** and paste to run the verification checklist and personalize

Keep all three prompts in the same chat window. Each step builds on the previous one.
